# snowboardjs
WebGLスノーボードゲーム

## 環境構築

### 必要な環境
- Bun 1.0以上

### セットアップ手順

```bash
# 依存関係のインストール
bun install

# 開発サーバーの起動
bun run dev
```

開発サーバーが起動したら、ブラウザで `http://localhost:3000` を開いてアプリケーションを確認できます。

### その他のコマンド

```bash
# プロダクションビルド
bun run build

# プロダクションサーバーの起動
bun run start

# リンターの実行
bun run lint
```
